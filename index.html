Conceitos Gerais e Flags Comuns

As flags são modificadores que você pode adicionar aos comandos do Avrae para personalizar suas ações. Elas começam com hífen (-) e permitem controlar diversos aspectos dos comandos.

-t

Define um ou mais alvos para o comando. Use para especificar quem será afetado por ataques, magias ou efeitos.

Exemplos: !attack "Espada" -t "Orc 1" !cast "Cura" -t "Jogador1" "Jogador2" !init effect "Goblin 3" "Medo"

-n

Especifica quantidade ou nome personalizado. Usada para criar múltiplas instâncias ou dar nomes a monstros/NPCs.

Exemplos: !init madd "Goblin" -n 5 !attack -n "Flecha Flamejante" !monster "Dragão" -n "Smaug"

-l

Define o nível de uma magia. Usada com o comando !cast para conjurar magias em níveis superiores.

Exemplos: !cast "Bola de Fogo" -l 5 !cast "Cura" -l 3 -t "Aliado" !cast "Raio de Gelo" -l 2

adv / dis

Rola com vantagem (adv) ou desvantagem (dis). Pode ser usado em rolagens de ataque, testes de perícia e salvaguardas.

Exemplos: !check percepção adv !attack "Arco" dis !save destreza adv

-rr

Repete um comando múltiplas vezes. Ideal para ataques múltiplos ou rolagens em sequência.

Exemplos: !attack "Adaga" -rr 2 !r 1d20 -rr 4 !check arcana -rr 3

-i

Ignora custos de recursos (como slots de magia). Útil para testes ou quando o Mestre permite gasto livre.

Exemplos: !cast "Teletransporte" -i !action surge -i !channel divinity -i

Dica Importante

Você pode combinar várias flags em um único comando! Por exemplo:

!cast "Bola de Fogo" -l 5 -t "Goblin 1" "Goblin 2" -i

Este comando conjura Bola de Fogo no 5° nível, nos Goblins 1 e 2, ignorando o custo do slot de magia.

Comandos Essenciais

Dados

!roll / !r

Realiza rolagens de dados no formato XdY (X dados de Y lados). Aceita modificadores matemáticos.

Exemplos:
!r 1d20 // Rola um d20 !r 2d6+3 // Rola dois d6 e soma 3 !r 1d20+5 adv // Rola com vantagem
Dados

!check

Realiza um teste de perícia usando sua ficha de personagem. O bot calcula automaticamente os modificadores.

Exemplos:
!check percepção !check furtividade adv // Com vantagem !check persuasão -dc 15 // Define CD
Combate

!init

Comandos para gerenciar a iniciativa durante combates. Permite adicionar jogadores, NPCs e monstros.

Exemplos:
!init begin // Inicia um novo combate !init join // Entra na iniciativa !init madd "Orc" -n 3 // Adiciona 3 orcs !init next // Avança para o próximo turno
Combate

!attack

Realiza um ataque usando uma arma da sua ficha. Calcula acerto e dano automaticamente.

Exemplos:
!attack "Espada Longa" !attack "Arco Longo" -t "Goblin 1" // Define alvo !attack "Adaga" -rr 2 // Dois ataques
Personagem

!character / !char

Comandos para gerenciar sua ficha de personagem. Permite visualizar, alternar e configurar personagens.

Exemplos:
!char // Mostra seu personagem atual !char list // Lista seus personagens !char switch "Nome" // Troca de personagem
Personagem

!hp

Gerencia os pontos de vida do seu personagem. Permite adicionar, remover ou definir valores.

Exemplos:
!hp // Mostra HP atual !hp -5 // Toma 5 de dano !hp +10 // Cura 10 pontos !hp set 25 // Define HP para 25
Magias

!cast

Conjura uma magia da sua ficha. Gasta slots automaticamente e calcula efeitos.

Exemplos:
!cast "Bola de Fogo" !cast "Cura" -l 3 // Conjura no nível 3 !cast "Raio de Gelo" -t "Goblin 1" "Goblin 2" // Define alvos
Utilitários

!rest

Realiza descansos (curto ou longo) para recuperar recursos e pontos de vida.

Exemplos:
!rest short // Descanso curto !rest long // Descanso longo !rest long -h // Recupera todos os pontos de vida
